# Todo App

## Description

Une application Todo simple construite avec Vue.js, Express et MongoDB, déployée via Docker. Ce dépôt sert de base pour évaluer les compétences techniques des développeurs candidats.

## Prérequis

- Docker
- Docker Compose

   cd todo-app
